# waka projects

List all projects you have logged time against.

## Usage

```sh
waka projects [OPTIONS]
```

## Options

| Flag             | Description                             |
| ---------------- | --------------------------------------- |
| `--format <fmt>` | Output format: `text` (default), `json` |
| `--color <when>` | Color mode: `auto`, `always`, `never`   |

## Examples

```sh
waka projects
waka projects --format json
```
